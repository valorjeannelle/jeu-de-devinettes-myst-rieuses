# Devinettes Mystérieuses

Un petit jeu de devinettes locales, conçu pour être joué en ligne. L’objectif est simple : deviner, réfléchir, et s’amuser avec des énigmes inspirées du quotidien.

## Fonctionnement

Le jeu fonctionne directement dans le navigateur, sans téléchargement.  
Une connexion internet est nécessaire pour jouer.

## Caractéristiques

- Interface légère et accessible
- Devinettes 100% locales, sans API
- Navigation simple et responsive
- Code organisé et facile à adapter

##  Accès au jeu

https://devinettesmysterieuses.netlify.app/

## Technologies utilisées

- HTML
- CSS
- JavaScript

##  Structure du projet

Le projet est organisé de manière à faciliter la lecture, la maintenance et l’évolution du code :

/devinettes-mysterieuses/
├── index.html            → Page principale du jeu
├── style.css             → Feuille de style pour l’interface
├── script.js             → Logique du jeu et interactions
├── data/
│   └── devinettes.json   → Liste des devinettes locales (hors ligne)
├── assets/
│   ├── images/           → Illustrations et icônes
│   └── sounds/           → Effets sonores (optionnels)
└── README.md             → Documentation du projet

Chaque fichier a un rôle bien défini pour garantir une expérience fluide et une adaptation facile à d’autres contextes ou langues.

##  Remarques finales

Ce projet est conçu pour célébrer la créativité locale et l’accessibilité numérique.  
Merci pour votre curiosité — amusez-vous bien !
